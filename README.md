## Currency Types

- **Gold** - Premium currency for serious players (1,000 - 10,000 Gold)
- **Silver** - Daily use currency (2,000 - 25,000 Silver)
- **Diamond** - Exclusive premium currency (100 - 1,000 Diamonds)

## Usage

Simply open `index.html` in a web browser to view the shop. The website is fully functional with:
- Product browsing
- Add to cart functionality
- Cart management
- Checkout preparation (ready for Tebex integration)

## Future Integration

This shop is designed to be easily integrated with Tebex for payment processing. The checkout functionality is prepared to connect with Tebex payment gateway.
